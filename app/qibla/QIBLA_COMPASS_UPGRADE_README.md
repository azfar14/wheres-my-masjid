# Qibla Compass Upgrade

This upgrade improves the Qibla page for easier mobile use.

## What changed

- Compass now feels more like a normal compass.
- Added a clear Kaaba marker on the Qibla direction.
- Added a top guide marker so users can turn until the Kaaba reaches the top.
- Simplified the wording to be easier for elderly users.
- Added a 3 step quick guide.
- Improved mobile layout so buttons, cards, and grids stack cleanly without overlap.
- Added stronger responsive styles for small phones.

## Files changed

- `app/qibla/page.tsx`
- `app/globals.css`

## How to apply

If you are merging manually, replace those two files in your project.

Then run:

```bash
npm run dev
```

or redeploy on Vercel.
