# Chennai starter masjid sources

The five starter listings were added for MVP testing so that the app can show real Chennai masjid names and coordinates without manually creating Firestore fields one by one.

The jamaat timings are placeholders only and must be verified locally before public launch.

Included listings:

1. Thousand Lights Mosque — Anna Salai / Thousand Lights, Chennai — coordinates 13.05500, 80.25500.
2. Triplicane Big Mosque / Wallajah Mosque — Triplicane High Road, Chennai — coordinates 13.06, 80.27.
3. Mamoor Mosque / Masjid-e-Mamoor — Angappan Street, George Town, Chennai — coordinates 13.0945, 80.2901.
4. Periamet Mosque — corner of Sydnehams Road and Vepery High Road, Periamet — coordinates 13.08468, 80.27018.
5. Hafiz Ahmad Khan Mosque / Ice House Mosque — Quai-de-Millath Road near Vivekanandar Illam, Chepauk — coordinates 13.053128, 80.27348.
